Privacy Policy

Beyond Copy-Paste: AI-Resilient Teaching Toolkit — Chrome Extension

Last updated: March 12, 2026

Summary: This extension does not collect, store, transmit, or share any user data whatsoever. It runs entirely in your browser with no server communication.

What Data We Collect

None. This extension collects zero user data. No personal information, no browsing history, no usage analytics, no cookies, and no telemetry of any kind.

How the Extension Works

Beyond Copy-Paste is a side-panel reference tool for educators. All content — lesson transformation prompts, assignment examples, grading guides, classroom norms, and diagnostic tools — is bundled locally within the extension. Everything runs in your browser. Nothing is sent to any external server.

When you use the "Open in Claude" button, you are choosing to navigate to claude.ai with your generated prompt. This is a standard browser navigation initiated by you — the extension does not transmit any data itself.

Permissions

This extension uses only the sidePanel permission, which allows it to display a sidebar in Chrome. It does not request access to your tabs, browsing activity, history, bookmarks, or any other browser data.

Third-Party Services

This extension does not integrate with, send data to, or receive data from any third-party services, APIs, or analytics platforms.

Data Storage

Any text you enter into the extension's forms (such as lesson descriptions or assignment text) exists only in your browser's active memory while the side panel is open. It is not saved, stored, or persisted anywhere. Closing the side panel clears all entered content.

Children's Privacy

This extension is designed for educators. It does not collect data from any user, including children under 13.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL with a new "Last updated" date. Material changes will be noted in the extension's update description on the Chrome Web Store.
